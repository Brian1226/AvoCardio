## Installation

Install these (preferably the latest versions)
```bash
python3
pip3
```

Clone the repository
```bash
git clone https://github.com/Brian1226/AvoCardio.git
```

Install all dependencies
```bash
pip3 install -r requirements.txt
```

Run the application
```bash
python3 main.py
```
